Chronos-2 Universal Forecasting WebUI Demo
=========================================

本仓库复用 `chronos-2-WebUI` 中的 Gradio 前端骨架与可视化布局，用于展示“从单一到通用预测”这一能力在不同时间序列任务中的交互形式。所有代码与截图均面向零样本与小样本的时间序列应用场景，强调通过统一界面承载多种任务配置的可视化效果，而不在本仓库内部下载或存储任何真实模型权重。读者如需了解完整的技术背景与学位论文式说明，可以参考主仓库中的中文 README 文档，其中系统介绍了 Chronos-2 在模型架构、概率建模与工程实现层面的设计思想。

本仓库中的 `app.py`、`webui_mock.html` 以及两张 PNG 截图文件与主仓库保持一致，方便在不同项目名称与说明文字下进行快速复用与二次开发。对于需要在多个业务条线之间共享同一套可视化前端的团队而言，这种“统一代码、多仓库命名”的组织方式，可以在保持技术栈一致性的同时，对外体现出更加清晰的应用定位。
